# Unified Pricing Architecture

## Overview
This document describes the new unified pricing system that synchronizes costs between LiteLLM proxy and the FastAPI backend, ensuring consistent billing across all touch points.

## Goals
1. Single source of truth for model pricing
2. Accurate cost tracking using LiteLLM's actual spend data
3. Configurable profit margins applied at LiteLLM level
4. Fail-safe: Block requests when pricing unavailable

## Data Flow

### 1. Pre-Call Cost Estimation (Backend)
```
User Request → Backend estimates cost from pricing.yaml
                    ↓
            Check user balance ≥ estimated_cost
                    ↓
            Reserve estimated amount
                    ↓
            Forward to LiteLLM
```

### 2. Actual Cost Processing (LiteLLM → Backend)
```
LiteLLM Proxy → Provider API
       ↓
   Response with headers:
   - x-litellm-response-cost: $0.000154 (base + margin)
   - usage: {prompt_tokens: 100, completion_tokens: 50}
       ↓
   Backend reads header
       ↓
   Deduct actual cost from reserved amount
       ↓
   Refund excess reservation
```

### 3. Frontend Display
```
Frontend → GET /billing/models
              ↓
          Returns pricing.yaml data
              ↓
          Display cost estimates
```

## Pricing Calculation Formulas

### Pre-Call Estimation (Backend)
```python
# For balance check before API call
estimated_cost = (estimated_input_tokens / 1000) * input_price_per_1k * (1 + margin) +
                 (estimated_output_tokens / 1000) * output_price_per_1k * (1 + margin)
```

### Post-Call Actual Cost (From LiteLLM)
```python
# Read from LiteLLM response header
actual_cost = float(response.headers.get('x-litellm-response-cost', 0))

# If header missing, fallback to local calculation
if not actual_cost:
    actual_cost = (prompt_tokens / 1000) * input_price_per_1k * (1 + margin) +
                  (completion_tokens / 1000) * output_price_per_1k * (1 + margin)
```

### User Billable Amount
```
User Charged = Actual Cost (already includes your margin)
Your Profit  = Actual Cost - Provider Cost
```

## Error Handling

### Missing Pricing Config
```python
if model not in PRICING:
    raise HTTPException(
        status_code=400,
        detail=f"Pricing not available for model '{model}'. Request blocked."
    )
```

### LiteLLM Header Missing
```python
actual_cost = response.headers.get('x-litellm-response-cost')
if actual_cost:
    actual_cost = float(actual_cost)
else:
    # Fallback to local calculation with warning
    logger.warning(f"x-litellm-response-cost missing for {model}, using local calc")
    actual_cost = billing_manager.calculate_cost(model, tokens_in, tokens_out)
```

### Pricing Reload Failure
```python
@admin_router.post("/reload-pricing")
async def reload_pricing():
    try:
        new_pricing = load_pricing_config()
        validate_pricing(new_pricing)
        global PRICING
        PRICING = new_pricing
        return {"status": "success", "models_loaded": len(new_pricing)}
    except Exception as e:
        raise HTTPException(500, f"Failed to reload pricing: {e}")
```
